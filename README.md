# Java Lottery Game 

A console-based lottery game built in Java.  
The player can deposit money, place bets, choose difficulty levels, and win or lose based on random draws.

## Features
- Secure numeric input (prevents letters & symbols)
- Wallet system (deposit, bet, win/lose)
- Difficulty levels (Easy / Normal / Difficult)
- Menu-driven interface
- Loading animation for better user experience

## Technologies Used
- Java
- Scanner
- Random numbers
- Console-based UI

## How to Run
1. Compile the program:
   lottery.java
2. Run the program:
   java lottery
